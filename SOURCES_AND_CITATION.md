# Sources and Citation

This project uses the Python package `questplus` for Bayesian adaptive psychometric estimation (QUEST+ workflow).

## 1) Software source (package used in this project)

- Repository: [https://github.com/hoechenberger/questplus](https://github.com/hoechenberger/questplus)
- PyPI package: [https://pypi.org/project/questplus/](https://pypi.org/project/questplus/)
- License: GNU General Public License v3.0 (GPL-3.0)

Suggested software citation:

Höchenberger, R. *questplus* (Python package). GitHub repository. Available at: https://github.com/hoechenberger/questplus

## 2) Method citation (QUEST+ paper)

Recommended citation for the QUEST+ method:

Watson, A. B. (2017). QUEST+: A general multidimensional Bayesian adaptive psychometric method. *Journal of Vision, 17*(3), 10. https://doi.org/10.1167/17.3.10

Optional foundational citation (original QUEST):

Watson, A. B., & Pelli, D. G. (1983). QUEST: A Bayesian adaptive psychometric method. *Perception & Psychophysics, 33*(2), 113-120. https://doi.org/10.3758/BF03202828
