# Hardware Design & Specifications

This folder contains the mechanical and electronic design files for the TDT research apparatus.

##   File Inventory

m5stack.png          | Graphical schematic illustrating the electronic connections and signal flow.
m5stack_readme.txt   | A technical map of the M5Stack firmware and hardware configuration, including wiring diagrams and GPIO pin mapping (Pinout).
stand.io             | The 3D assembly model of the LEGO-based stand. This file can be opened, edited, and rendered using BrickLink Studio 2.0.
stand.png            | A high-resolution render of the LEGO apparatus for quick visual reference.
stand_readme.txt     | Additional information about the LEGO-based stand.
INSTRUCTIONS_EN.md   | Comprehensive test instructions for the participant.
INSTRUCTIONS_PL.md   | Comprehensive test instructions for the participant.
