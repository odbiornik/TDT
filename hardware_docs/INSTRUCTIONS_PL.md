# Instrukcja dla osoby badanej

### 1. Sygnał startu
Badanie rozpocznie się od wyświetlenia sekwencji świateł kontrolnych: 
**czerwone** → **żółte** → **zielone**. 
Gdy zobaczysz zielone światło, oznacza to, że za chwilę pojawią się pierwsze bodźce.

### 2. Twoje zadanie
W trakcie testu będziesz widzieć bardzo krótkie białe błyski. Twoim zadaniem jest zdecydować:
* Czy zobaczyłeś/aś **jeden** błysk?
* Czy zobaczyłeś/aś **dwa oddzielne** błyski następujące szybko po sobie?

### 3. Przyciski odpowiedzi
Po każdym bodźcu naciśnij odpowiedni przycisk:
* 🔵 **Niebieski przycisk** = widziałem/am **DWA** oddzielne błyski.
* 🔴 **Czerwony przycisk** = widziałem/am **JEDEN** błysk (lub zlały się w jedno).

### 4. Ważne wskazówki
* **To nie jest test refleksu:** Nie musisz się spieszyć. Liczy się dokładność i Twoje rzeczywiste wrażenie.
* **Zaufaj intuicji:** Odpowiadaj naturalnie, zgodnie z pierwszym wrażeniem. Nie analizuj zbyt długo każdego błysku.
* **Decyduj mimo niepewności:** Jeśli nie masz 100% pewności, wybierz opcję, która wydaje Ci się bardziej prawdopodobna.

### 5. Koniec badania
Zakończenie testu zostanie zasygnalizowane **stałym czerwonym światłem** obu lampek.
